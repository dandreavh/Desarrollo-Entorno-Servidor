Definición

Express es una infraestructura de aplicaciones web Node.js (API) que proporciona un conjunto sólido de características para las aplicaciones.

Primeros pasos

  • Instalar Nodejs
  • Crear el "package.json" con npm init
  • Instalar Express y guardar las dependencias con npm install express --save
