Definición

Node.js es un entorno en tiempo de ejecución multiplataforma, de código abierto, para la capa del servidor basado en JavaScript, asíncrono, con una arquitectura orientada a eventos.

Primeros pasos

  • Instalar desde la web oficial de Node.js
  • Tras instalar Express, iniciar el servicio con node app.js
  • Comprobar que esté escuchando entrando en el navegador con localhost:8080, por ejemplo, que ha sido el puerto elegido
